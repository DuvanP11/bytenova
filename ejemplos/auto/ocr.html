¿Para qué áreas?

Empresas con onboarding de clientes / proveedores / empleados que reciben cédulas, facturas o contratos diariamente y los validan a mano. Fintechs, aseguradoras, inmobiliarias, RRHH, KYC.

El sistema usa OCR de última generación (Google Vision, AWS Textract) + verificación cruzada contra Registraduría (cédulas), DIAN (RUT/facturas), Procuraduría (antecedentes). Marca como "sospechoso" cuando los datos extraídos no coinciden con las fuentes oficiales.

Una persona que antes validaba 30 cédulas al día, ahora supervisa 300 (solo revisa las que el sistema marca dudosas).

QUÉ INCLUYE

  • OCR de cédulas, RUT, facturas, contratos
  • Validación contra Registraduría / DIAN
  • Detección de documentos alterados
  • Bandeja de excepciones a revisar
  • API para integrar a su sistema
  • Auditoría completa (quién validó qué)
  • Hasta 5.000 docs/mes incluidos
